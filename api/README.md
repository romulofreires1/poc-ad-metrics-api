## UI

{host}:3000/swagger

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
